# 📊 Excel-Superstore-Sales-Performance-Dashboard

Developed an interactive sales analytics dashboard in Microsoft Excel to analyze business performance across profitability, customer segments, category analysis and regional trends using Superstore sales data.

📌 Project Overview

This project analyzes Superstore sales data using Excel to build an interactive dashboard that tracks profit, losses, customer segmentation, Monthly trends and sales performance across categories and regions.

## Dataset Information

Source: SuperStoreOrders.xlsx

Records Analyzed: 6592

Features Included:

  - Order Details
  - Customer Information
  - Product Categories
  - Sales / Profit / Quantity
  - Region / Market Analysis

## Objectives
- Analyze total profitability, loss and total orders 
- Track monthly profit trends
- Segment customers by value
- Compare performance across categories and regions

## Dashboard KPIs
- Total Profit
- Total Loss
- Number of Orders
  
## Visualizations Used
- KPI Cards
- Profit by Category Bar Chart
- Profit by Region Bar Chart
- Monthly Profit Trend Line Chart
- Customer Segment Pie Chart

## Key Insights
- Central region contributes highest revenue.
- Technology is the highest profitable category.
- Profit peaks early in February-March and then steadily declines through the year,hitting itslowest point in November.
- High-value customers contribute the majority of profit about ~84%.
  
## Tools Used
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - Conditional Formatting
  - Dashboard Design Techniques
 
## Files Included
 `SuperStoreOrders.xlsx`

      Sheet 1 - SuperStoreOrders → Raw Data

      Sheet 2 - Cleaned_data → Cleaned Data
